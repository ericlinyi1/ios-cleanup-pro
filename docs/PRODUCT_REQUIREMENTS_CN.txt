# iOS Cleanup Pro - 需求规格说明书 (PRD)

## 1. 项目愿景
开发一款极简、极速且安全的 iOS 原生应用，利用 AI 技术帮助用户智能清理手机空间。

## 2. 核心功能模块

### 2.1 权限与引导 (Authentication)
- **需求**：用户首次打开应用需明确授权相册访问。
- **功能**：支持“完整访问”权限请求，提供隐私说明页面，支持引导跳转至系统设置。

### 2.2 重复照片识别 (Duplicate Detection)
- **需求**：精准查找完全一致的副本。
- **功能**：通过文件哈希（Byte-for-byte）比对，快速聚合重复组。支持“一键保留最佳版本”。

### 2.3 相似照片聚类 (Similar Photos)
- **需求**：识别连拍、构图相似的照片。
- **功能**：利用 Apple Vision Framework 生成 Feature Print，计算余弦相似度。支持手动调节敏感度阈值。

### 2.4 屏幕截图清理 (Screenshots)
- **需求**：专门处理由于截图堆积造成的存储压力。
- **功能**：自动过滤 .photoScreenshot 子类，支持按应用来源或日期批量删除。

### 2.5 划动清理交互 (Swipe-to-Clean)
- **需求**：提供有趣的决策体验。
- **功能**：实现 Tinder 风格的 UI。左划加入待删除队列，右划保留。

### 2.6 视频压缩与管理 (Video & Collection)
- **需求**：减少大视频占用的空间。
- **功能**：识别 100MB 以上视频，支持 HEVC (H.265) 硬件压缩。支持将照片批量移动至新建或现有的 Collection 文件夹。

## 3. 验收标准 (Definition of Done)
- 所有功能必须通过真机/模拟器 E2E UI 测试。
- 每一个功能点的结项必须在 GitHub Issue 中附带实机截图证据。
- 文档必须与代码逻辑 100% 同步。
